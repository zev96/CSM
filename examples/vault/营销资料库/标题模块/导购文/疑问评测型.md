# 疑问评测型 标题公式

"开头抛问 + 后段自答"的经典结构，比纯导购更有"评测"的客观感。
关键词紧跟"怎么选/怎么挑"承接搜索意图，年份 + 数量给可信度。

## 适用场景

- 关键词没有明确品牌指向，需要先做品类对比
- 用户处在调研阶段，想看客观盘点
- 内容偏中长，有横向对比表 / 测评维度
