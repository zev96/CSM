# 标题模块

`csm_core/title/generator.py` 在生成文章前会扫这里抓公式 + 样例，喂给 LLM 出 3 个候选标题。
本目录是**示例骨架**，把整棵 `营销资料库/标题模块/` 拷进你自己的 vault 即可生效。

## 目录约定

```
营销资料库/标题模块/
  导购文/
    一问一答型.md
    避坑型.md
    疑问评测型.md
  对比文/
    ...
  单品文/
    ...
  长文/
    ...
```

## 单条标题公式的 frontmatter

最少四个字段：

| 字段 | 说明 | 必需 |
|---|---|---|
| `标题类型` | 这条公式的"风格名"，会作为 prompt 里的分组 head 给 LLM 看 | 是 |
| `适用模板类型` | 字符串或列表（`[导购文, 对比文]`），决定哪些模板会调用这条公式 | 是 |
| `公式` | 原始公式串，带占位符（`[关键词] [疑问词] [时效词] [数量] [利益]`） | 是 |
| `示例` | 真实样例标题列表 — few-shot，越具体越好 | 强烈建议 |
| `插槽` | 槽位词典（`疑问词: [哪款, 哪种]`），目前仅用于人类参考；LLM 自由选词 | 可选 |

## 新增标题类型

1. 在对应 `<模板类型>/` 目录下加一个新 `.md`
2. frontmatter 里 `适用模板类型` 写上要挂的模板分类
3. 重新打开 CSM 应用 / 重新生成文章 — 即时生效，不用动代码

## 怎么调试

如果 LLM 出的标题不对劲：

- 确认 vault 路径在「设置 → 存储路径 → 素材库」里配好
- 确认 `适用模板类型` 拼写跟 [`csm_core.template.schema.TemplateType`](../../../csm_core/template/schema.py) 一致（导购文 / 对比文 / 单品文 / 长文）
- LLM 兜不住时会回退到 `f"{关键词}怎么选？看完这篇就不踩坑"`，看到这句说明前面的链路某处坏了
